Note:

You may use the applicable portion of the script in C:\Training\Lab 22 - Locking

The EnableDisableKeys() script function receives three Boolean parameters to disable the

Alt, Esc, and Win keys, in that order: 0 (false) enables the key; 1 (true) disables the key.

65. Click OK to validate, save, and close the Data Change script.

Tagname[.field]:

$AccessLevel

Body:

{ Menu Access }

IF $AccessLevel > 5000 THEN

{ Allow menu access }

Hide "MenuBarCover";

EnableDisableKeys( 0, 0, 0);

ELSE

{ Deny menu access }

Show "MenuBarCover";

EnableDisableKeys( 1, 1, 1);

ENDIF;

Data Change Scripts editor with $AccessLevel tag, showing the updated script with EnableDisableKeys() function: IF $AccessLevel > 5000 THEN Hide "MenuBarCover"; EnableDisableKeys(0,0,0); ELSE Show "MenuBarCover"; EnableDisableKeys(1,1,1); ENDIF; — Disables Alt, Esc, and Win keys for low-access users.
