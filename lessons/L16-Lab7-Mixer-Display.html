Lesson 16: Lab 7 – Creating the Mixer Display

Source: AVEVA InTouch for System Platform 2023 Training Manual, Module 3, Lab 7 (pages 181–201)

In this lab, you will build the complete Mixer display by embedding graphics from contained objects, adding text elements with Value Display animations, drawing connectors between equipment, and replacing the placeholder graphic in the ContentDisplay window.

Lab Objectives: Upon completion of this lab, you will be able to:
• Add a symbol in an automation object
• Embed graphics from relative references
• Add Text elements and configure Value Display animations
• Apply element styles to elements
• Use connectors and connection points
• Change element display order
• Replace a graphic in a frame window

16.1 Creating the MixingProcess_Detailed Symbol

Reference: Module 3, Lab 7 (pages 180–181)

You begin by creating a new symbol inside the $Mixer template. This symbol — named MixingProcess_Detailed — will contain all the equipment graphics for the mixer process. Open the $Mixer template from the Templates tab, click Add in the Content pane, name the symbol MixingProcess_Detailed, and give it the description "Includes symbols from all contained objects."

Next, you open the symbol for editing by clicking the Edit Content button (or double-clicking the symbol name in the Content pane). You will now embed the individual equipment graphics from the contained objects using relative references.

The MixingProcess_Detailed symbol in the Content pane, ready to open in the Graphic Editor (page 181)

16.5 Changing Element Display Order

Reference: Module 3, Lab 7 (page 186)

The Tank graphic currently appears on top of the other elements, covering the agitator and meters. To fix this, you need to send it to the back of the display order. With the Tank selected, right-click and select Order | Send To Back. The tank moves behind all other elements, and the Elements panel updates to show the new stacking order.

The Elements panel showing Tank at the bottom of the z-order after Send To Back (page 186)
The right-click context menu with Order | Send To Back selected (page 186)
Key Concept: The z-order (stacking order) in the Elements panel determines which elements appear on top when they overlap on the canvas. Use Order | Send To Back or Bring To Front to rearrange elements.

16.7 Straightening Connectors and Aligning Elements

Reference: Module 3, Lab 7 (pages 192–193)

The connectors may have bends that you want to remove. In the Elements panel, hold Ctrl and select Connector2, Connector3, Connector4, and Connector5. On the Properties tab, change the ConnectionType dropdown to Straight. This removes the bends from the selected connectors.

Selecting multiple connectors in the Elements panel for batch editing (page 192)
Setting ConnectionType to Straight to remove connector bends (page 192)

To align the pumps and valves, select Pump1 on the canvas, hold Shift and select Pump2, then use the arrow keys to nudge both pumps until the connectors between pumps and valves are straight. Adjust the valves if needed to ensure all connectors appear straight.

16.11 Introduction to Combo Box Controls

Reference: Module 3, Section 5 (pages 203–204)

Before moving on to the next lab, it is helpful to understand the Combo Box control, which you will use in Lab 8 to create a mixer selector. Combo Box controls allow users to select from a foldable list at runtime. There are three types:

TypeBehavior
SimpleUsers can type a value or select one using up/down arrow buttons
DropDownUsers can type a value or select one from the dropdown list
DropDownListUsers can only select from the predefined list — they cannot type custom values

You configure a Combo Box with a Combo Box animation, which is unique to this control type. The Static Values and Captions list defines the selectable options, and you can access runtime properties like Count (number of items) and NewIndex (index of the last added item) through scripting.

Knowledge Check

1. How do you embed a graphic from a contained object into a symbol on the canvas?

Click to reveal answer

Right-click the canvas and select Embed Industrial Graphic. In the Galaxy Browser, click the Relative References button to see contained objects, then select the desired relative reference (e.g., Me.Pump1\Pump_Detailed) and click OK.

2. Why do you need to use Order | Send To Back on the Tank graphic?

Click to reveal answer

The Tank graphic is placed on top of the agitator, level, and temperature graphics. Sending it to the back moves it behind those elements so they are visible inside the tank boundary.

3. What happens when you change a connector's ConnectionType to "Straight"?

Click to reveal answer

The bends in the connector are removed, making the connector a straight line between the two connection points. You may then need to adjust element positions to ensure the connectors align properly.

4. What does the Value Display animation with the Tag Name option show at runtime?

Click to reveal answer

It displays the tag name of the hosting object — for example, it shows "$Mixer" for the mixer template, allowing operators to identify which object the graphic represents.

5. Why does WindowMaker prompt you to confirm when replacing an embedded symbol in a frame window?

Click to reveal answer

Frame windows can only host one embedded symbol. Confirming the replacement removes the old symbol (e.g., Level_Detailed) and replaces it with the new one (MixingProcess_Detailed).
