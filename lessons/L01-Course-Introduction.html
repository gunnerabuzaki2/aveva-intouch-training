Lesson 1: Course Introduction

Source: AVEVA InTouch for System Platform 2023 Training Manual, Module 1, Section 1 (pages 1–6)

An overview of the course structure, learning objectives, and prerequisites for the AVEVA InTouch for System Platform training.

1.1 About This Course

Reference: Module 1, Section 1 (pages 1–2)

This course is a four-day, instructor-led training program designed to provide a comprehensive overview of the features and capabilities of AVEVA InTouch for System Platform. You will learn how to build and deploy visualization applications using InTouch, create graphics, visualize alarms and events, trend historical data, and implement application security. Hands-on labs reinforce the concepts covered throughout the course.

Key Concept: This training covers the full lifecycle of an InTouch visualization application — from initial setup through Galaxy creation, graphics development, alarm and trend configuration, security hardening, and web-based deployment.

1.2 Course Objectives

Reference: Module 1, Section 1 (page 1-3)

Upon completing this course, you will be able to:

  • Describe the AVEVA InTouch for System Platform visualization components
  • Create and run visualization applications from scratch
  • Understand and apply Situational Awareness design principles
  • Create and configure graphics using the Graphic Editor
  • Implement navigation frameworks for multi-screen applications
  • Configure and display alarm visualizations (real-time and historical)
  • Implement real-time and historical trending of process data
  • Apply security features to protect your visualization application
  • View and interact with graphics through a web browser via the Web Client

1.3 Course Structure and Modules

Reference: Module 1, Section 1 (pages 1-3 to 1-6)

The training manual is organized into eight modules, each building on the knowledge from previous sections:

1.4 Intended Audience

Reference: Module 1, Section 1 (page 1-3)

This course is designed for individuals who need to configure, modify, or maintain AVEVA InTouch for System Platform visualization applications — including system integrators, automation engineers, and operations personnel who work with industrial HMI/SCADA systems.

1.5 Prerequisites

Reference: Module 1, Section 1 (page 1-3)

Before taking this course, you should have knowledge of the following:

  • Industrial automation software concepts — familiarity with SCADA, HMI, and process control terminology
  • From the AVEVA Application Server product:
    • System Platform IDE navigation
    • AutomationObjects (object-oriented modeling)
    • Alarms of attributes
    • Historization of attributes
    • Security (authentication, roles, permissions)
    • Deployment model
    • Plant model (object hierarchies and areas)
    • QuickScript.NET scripting language
Tip: If you are new to Application Server, consider completing the AVEVA Application Server training first. It provides the foundational knowledge needed for this course.
